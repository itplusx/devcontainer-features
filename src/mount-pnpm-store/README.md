# Mount pnpm Store (mount-pnpm-store)

Sets pnpm store to ~/.pnpm-store and mounts it to a volume to share between multiple devcontainers. Patched copy of [joshuanianji/devcontainer-features](https://github.com/joshuanianji/devcontainer-features/tree/main/src/mount-pnpm-store) that works with pnpm >= 9 in non-interactive lifecycle shells.

## Example Usage

```json
"features": {
    "ghcr.io/itplusx/devcontainer-features/mount-pnpm-store:1": {}
}
```
